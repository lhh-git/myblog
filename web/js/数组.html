# 数组的定义

# 什么是数组？

数组是一种特殊类型的对象。在 JavaScript 中对数组使用 typeof 运算符会返回 "object"。 但是，JavaScript 数组最好以数组来描述。 数组使用数字来访问其“元素”。在本例中，person[0] 返回 Bill：

数组：
var person = ["Bill", "Gates", 62];

对象使用名称来访问其“成员”。在本例中，person.firstName 返回 John：

对象：
var person = {firstName:"John", lastName:"Doe", age:46};

# 访问数组元素

我们通过引用索引号（下标号）来引用某个数组元素。 这条语句访问 cars 中的首个元素的值：

var cars = ["Saab", "Volvo", "BMW"];
var name = cars[0];                     // Saab

# 改变数组元素

这条语句修改了 cars 中第一个元素的值：

var cars = ["Saab", "Volvo", "BMW"];
cars[0] = "Opel";                        // ["Opel", "Volvo", "BMW"]

# 访问完整数组

通过 JavaScript，可通过引用数组名来访问完整数组：

var cars = ["Saab", "Volvo", "BMW"];
document.getElementById("demo").innerHTML = cars;

# 数组的属性

# 1. length

Length属性表示数组的长度，即其中元素的个数。因为数组的索引总是由0开始，所以一个数组的上下限分别是：0和length-1。

var arr = [1, 2, 3]
arr[0]                                    // 1 
arr[arr.length - 1]                       // 3

Length属性可以清空数组

var arr = [1, 2, 3]
arr.length = 0
arr                                       // []
arr.length                                // 0

和其他大多 数语言不同的是，JavaScript数组的length属性是可变的，这一点需要特别注意。当length属性被设置得更大时，整个数组的状态事实上不 会发生变化，仅仅是length属性变大；当length属性被设置得比原来小时，则原先数组中索引大于或等于length的元素的值全部被丢失

var arr = [1, 2, 3]
arr.length                                // length: 3
arr.length = 6                            // length = 6; arr = [ 1, 2, 3, <3 empty items> ]
arr.length = 2                            // length = 2; arr = [1, 2]

length对象不仅可以显式的设置，它也有可能被隐式修改

var arr = [1, 2, 3]
arr[9] = 10
arr                                       // [ 1, 2, 3, <6 empty items>, 10 ] 
arr.length                                // 10

# 2. constructor

对创建数组对象的Array构造函数的引用

constructor属性是所有具有prototype的对象的成员。它们包括除Global和Math对象以外的所有JScript固有对象。constructor属性保存了对构造特定对象实例的函数的引用

function isArray(x) {
  return x.constructor.toString().indexOf("Array") > -1;
}

const arr = [1, 2, 3]
isArray(arr)                              // true
const obj = { name: 'jack' }
isArray(obj)                              // false

# 3. prototype

我们创建的每一个函数都有一个prototype（原型）属性，这个属性是一个指针，指向一个对象， 而这个对象的用途是包含可以由特定类型的所有实例共享的属性和方法。prototype方法能让我 们有能力向对象添加属性和方法。

给数组对象添加返回数组中最大元素值的方法。要完成这一点，声明一个函数，将它加入Array.prototype，并使用它

function array_max() {  
  var max = this[0];  
  for( var i = 1; i < this.length; i++ ) {  
    if( max < this[i] )  
    max = this[i];  
  }  
  return max;  
}  
 
Array.prototype.max = array_max;  
var x = new Array(1, 2, 3, 4, 5, 6);  
var y = x.max(); 
y                                           // 6

# 数组的方法

# push

push() 方法给数组末尾添加一个或多个元素，并返回新的长度

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.push("Lemon");                       // 向 fruits 添加一个新元素 (Lemon)

# pop

pop() 方法删除数组的最后一个元素，并返回删除的元素的值

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.pop();                               // 从 fruits 删除最后一个元素（"Mango"）

# shift

shift() 方法删除数组的第一个值，并返回删除的元素的值

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.shift();                             // 从 fruits 删除第一个元素 "Banana"

# unshift

unshift() 方法给数组开头添加一个或多个元素，并返回新的长度

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.unshift("Lemon");                    // 向 fruits 添加新元素 "Lemon"

# toString

toString() 方法把数组转换为数组值（逗号分隔）的字符串

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.toString();                          // Banana,Orange,Apple,Mango

# join

join() 方法将数组的元素组起一个字符串，以separator为分隔符，省略的话则用默认用逗号为分隔符

toString()与join()转换字符串区别： toString()只能转换为逗号分隔的字符串，json()可以指定分隔字符

var arr = [1, 2, 3]
arr.join()                                  // 1,2,3
arr.join('')                                // 123
arr.join(',')                               // 1,2,3
arr.join('|')                               // 1|2|3

# splice

splice() 方法向/从数组中添加/删除元素，然后返回被删除的元素。该方法会改变原始数组。

第一个参数（0）定义了应添加新元素的位置（拼接）。
第二个参数（2）定义应删除多少元素。
其余参数（[5, 6, 7], 8, 9）定义要添加的新元素。
splice() 方法返回一个包含已删除项的数组：

var arr = [1, 2, 3]
arr.splice(0, 2, [5, 6, 7], 8, 9)           // [ 1, 2 ]
arr                                         // [ [ 5, 6, 7 ], 8, 9, 3 ]

# concat

concat() 方法通过合并（连接）现有数组来创建一个新数组。不会更改现有数组。它总是返回一个新数组

var arr1 = ["Cecilie", "Lone"];
var arr2 = ["Emil", "Tobias", "Linus"];
arr1.concat(arr2, 3)                        // [ 'Cecilie', 'Lone', 'Emil', 'Tobias', 'Linus', 3 ]

# slice

slice() 方法用数组的某个片段切出新数组。创建新数组。它不会从源数组中删除任何元素

slice() 可接受两个参数，比如 (1, 3)。
该方法会从开始参数选取元素，直到结束参数（不包括）为止。
返回一个包含已删除项的数组：
var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
fruits.slice(1, 3);                         // [ 'Orange', 'Lemon' ]
fruits                                      // [ 'Banana', 'Orange', 'Lemon', 'Apple', 'Mango' ]

如果结束参数被省略，如：slice(1) 会切出数组的剩余部分
返回一个包含已删除项的数组：
var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
fruits.slice(1);                            // [ 'Orange', 'Lemon', 'Apple', 'Mango' ]

# indexOf

indexOf() 方法在数组中搜索元素值并返回其位置。如果未找到项目，Array.indexOf() 返回 -1。

var fruits = ["Apple", "Orange", "Apple", "Mango"];
var a = fruits.indexOf("Apple");            // a = 0
如果项目多次出现，则返回第一次出现的位置

array.indexOf(item, start)
item	必需。要检索的项目。
start	可选。从哪里开始搜索。负值将从结尾开始的给定位置开始，并搜索到结尾。

# lastIndexOf

Array.lastIndexOf() 与 Array.indexOf() 类似，但是从数组结尾开始搜索。

var fruits = ["Apple", "Orange", "Apple", "Mango"];
var a = fruits.lastIndexOf("Apple");        // a = 2

# includes

includes(指定值, 起始位置(负数表示倒数))：判断一个数组是否包含一个指定的值。

[1, 2, 3].includes(2)                       // true
[1, 2, 3].includes(4)                       // false
[1, 2, NaN].includes(NaN)                   // true
[1, 2, 3].includes(3, 3)                    // false

# lastIndex

lastIndex(item)：返回一个指定元素在数组中最后出现的位置，没有则没有-1。

# ...

展开语法(Spread syntax), 可以在函数调用/数组构造时, 将数组表达式或者string在语法层面展开；还可以在构造字面量对象时, 将对象表达式按key-value的方式展开。(译者注: 字面量一般指 [1, 2, 3] 或者 {name: "mdn"} 这种简洁的构造方式)

const arr = [1, 2, 3]
let arr2 = [4, ...arr]
console.log(arr2)                           // [ 4, 1, 2, 3 ]

const arr = [1, 2, 3]
let arr2 = [...arr]
arr2.push(4)
console.log(arr)                            // [ 1, 2, 3 ]
console.log(arr2)                           // [ 1, 2, 3, 4 ]
展开语法和 Object.assign() 行为一致, 执行的都是浅拷贝(只遍历一层)。不可对多维数组进行深拷贝

# 冒泡排序

冒泡排序 (opens new window)

var arr = [0, 1, 23, 12, 2];
function sortBubbling(arr) {
	for( var i = 0; i < arr.length; i++ ){
		for( var j = 0; j < arr.length - 1 - i; j++ ){
			if( arr[j] > arr[j + 1] ){
				var tim = arr[j];
				arr[j] = arr[j+1];
				arr[j+1] = tim;
			}
		}
	}
	return arr;
}
console.log( sortBubbling(arr) )           // [ 0, 1, 2, 12, 23 ]

# break, continue, return

相同之处：

三个都会将此时进行的语句停止。

不同之处：

1、break：执行break操作，跳出所在的当前整个循环，到外层代码继续执行。
2、continue：执行continue操作，跳出本次循环，从下一个迭代继续运行循环，内层循环执行完毕，外层代码继续运行。
3、return：执行return操作，直接返回函数，所有该函数体内的代码（包括循环体）都不会再执行。
4、使用的语句环境不一样，break和continue是用在循环或switch语句中，return是用在函数语句中。

# for, for in, for of, while, do while

方法的回调函数中支持return返回值
return: 支持return阻断当前循环与跳出当前函数
break: 支持break跳出所在的当前整个循环
continue：支持continue操作，跳出本次循环，从下一个迭代继续运行循环，内层循环执行完毕，外层代码继续运行。

# for, while, do while

const arr = [1, 2, 3, 4, 5]
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]) // 1 2 3 4 5
}

let count = 0
while(count < 5) {
  count += 1
  console.log(count)   // 1 2 3 4 5
}

let count = 0
do {  //循环条件
  count += 1
  console.log(count) // 1
} while (count < 0)

for和while是先判断再执行
do...while是先执行一次再判断继不继续
所以for和while语句块可能一次都不执行，但是do...while至少会执行一次

# every

every()是对数组中的每一项运行给定函数，如果该函数对每一项返回true,则返回true

  })

const arr = [
  {
    id: 0,
    name: 'liu'
  }, 
  {
    id: 1,
    name: 'hai'
  }, 
  {
    id: 2,
    name: 'hong'
  }
]

function init(data) {
  console.log(data)      						// { id: 0, name: 'liu' } { id: 1, name: 'hai' }
  return data.name === 'hai'
}
console.log(arr.findIndex(init))  				// 1
